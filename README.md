# pokelib

Para iniciar o projeto basta clonar em seu PC, instalar as dendencias com:

```sh
# Recomendado
$ yarn
```
ou

```sh
$ npm install
```

e então

```sh
$ yarn start
  ou 
$ npm start
```

O projeto utiliza o boilerplate https://github.com/flexdinesh/react-redux-boilerplate com algumas pequenas adaptaçãoes
